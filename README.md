# 자바 공부하는 레포
시간은 없는데 공부할 건 많다!! 미루지 말고 부지런히, 하루에 한시간씩이라도 꾸준히 해보자 💪🔥
2023년 1월 26일 스타트

### 계기
- main이 어떻게 돌아가는지 static을 왜 쓰는지 메모리는 뭔지 기초지식도 없으면서
- 프레임워크 사용할 줄 안다고 자바 할 줄 안다고 말하고 다니고
- 이대로는 그냥저냥 스프링부트 쓸 줄 아는 코더가 될 것 같고
- 누가 자바에 대해 물어봤을 때 대답 못하는 쪽팔린 상황을 피하기 위해 공부를 시작한다...

### 공부방법
- 공부한 내용은 깃헙 이슈에 document로 기록하기
- 실습이 필요한 내용이 있다면 실습 후 해당 이슈에 커밋 (공부도 하고 잔디도 심고 개이득인부분)
- 문서 정리에 치우쳐서 정작 공부는 미루지 말기 / 정리 대충하기
- 자주 써봤다고 자만하지 않기
  - 많이 해본 부분이당 ㅎㅎ 대충 하고 넘어가자 금지
  - 익숙한 개념이면 스스로에게 질문해보고 검색해 볼
